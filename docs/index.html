Belajar Git

Git ialah sesuatu yang dinamakan version-control system. Version-control system ialah sebuah sistem untuk melihat perubahan pada fail-fail yang ada di komputer, dan untuk menguruskan perubahan-perubahan tersebut jika fail-fail tersebut perlu diubah oleh beberapa pengguna.

Tujuan utama Git adalah untuk melihat perubahan yang berlaku pada code. Jadi, kita boleh menggunakan Git untuk melihat apa yang kita ubah sekiranya sesuatu yang tidak diingini berlaku pada application kita.

Dalam tutorial ini, kita akan belajar asas penggunaan Git, penggunaan Git dalam kumpulan, berserta tips dan tricks semasa menggunakan Git.

Tunggu apa lagi, Jom Belajar Git!
