Installation

Git merupakan console application. Jadi, kita perlu menggunakan terminal atau command prompt untuk menggunakan Git. Kepada yang biasa menggunakan Linux memang akan biasa dengan console application. Kepada yang menggunakan Windows, anda boleh install mana-mana GUI client untuk Git.

Klik sini untuk lihat senarai GUI client

Harap maaf. Saya tidak ada pengalaman menggunakan GUI client untuk Git. Tutorial ini saya akan utamakan penggunaan Git melalui terminal.

Kepada yang menggunakan Linux, anda boleh lihat sama ada Git sudah install dengan menaip

git --version

di terminal. Jika terpapar versi Git bermaksud Git sudah install di komputer. Jika tidak, anda boleh install dengan menaip

sudo apt install git
